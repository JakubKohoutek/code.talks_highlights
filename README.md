## Getting Started

Run

```bash
npm install
npm start
```

Open a browser and hit [http://localhost:3210](http://localhost:3210), and you're ready to roll.

## Build & Deployment

Building the dist version of the project:

```bash
npm run build
```

If you want to deploy the slideshow to surge, run:

```bash
npm run deploy
```
